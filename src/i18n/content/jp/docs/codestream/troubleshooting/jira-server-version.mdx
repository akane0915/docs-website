Jira Server のほとんどのページでは、ページのフッターにバージョン番号が表示されています。以下はその例です。

![Jiraサーバーのバージョン](/images/JiraServerVersion.png)

ほとんどのページでは、この情報がフッターにあるはずです。たとえば、デフォルトのシステムダッシュボードですが、 **Jira について** **Help** メニューの下にあるページをチェックしてみることもできます。
