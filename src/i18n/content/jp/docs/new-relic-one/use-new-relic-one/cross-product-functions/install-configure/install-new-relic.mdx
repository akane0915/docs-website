[New Relicアカウントにサインアップ](https://newrelic.com/signup)（永久無料）し、当社のモニタリングサービスをインストールすると、[データの処理](/docs/data-apis/get-started/nrdb-horsepower-under-hood)を開始できます。

[New Relicのインスタントオブザーバビリティクイックスタート](https://newrelic.com/instant-observability/)を使用して、早速始めましょう。または、[ガイド付きインストール](/docs/new-relic-one/use-new-relic-one/cross-product-functions/install-configure/new-relic-guided-install-overview/)を使用してください。

New Relicモニタリングサービスをインストールする方法については、こちらのリンクをご覧ください。

* [APM](#apm-install)
* [ブラウザ](#browser-install)
* [インフラストラクチャ](#infra-install)
* [モバイル](#mobile-install)
* [合成モニター](#synthetics)
* [サードパーティのテレメトリーサービスのインテグレーション](/docs/data-ingest-apis/get-data-new-relic/new-relic-sdks/telemetry-sdks-send-custom-telemetry-data-new-relic#external-data)
* [データインジェストAPI](/docs/data-ingest-apis/get-data-new-relic/getting-started/introduction-new-relic-data-ingest-apis-sdks)（メトリクス、イベント、ログ、トレース）

## トラブルシューティング

New Relic UIでデータの表示を開始するには、エージェントをインストールし、トラフィックを生成してから、数分待つ必要があります。データが表示されない場合は、[トラブルシューティングプロセス](/docs/agents/manage-apm-agents/troubleshooting/not-seeing-data)に従ってください。
